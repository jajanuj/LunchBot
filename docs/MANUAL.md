## 1. 系統概覽

LunchBot 是一套公司內部午餐訂購管理系統，整合 LINE 群組推播與員工 LIFF 點餐頁面，讓助理可以統一管理每日菜單、彙整訂單、產生對帳清單與薪資扣款紀錄。

### 整體流程

```
助理建立菜單
    ↓
推播菜單至 LINE 群組
    ↓
員工透過 LINE LIFF 點餐（或助理代客點餐）
    ↓
截止時間到 → 系統自動結單
    ↓
助理查看叫貨清單、個人對帳清單
    ↓
推播叫貨清單至 LINE 群組 / 匯出 CSV
    ↓
月底產生薪資扣款紀錄
```

### 角色說明

| 角色 | 使用功能 |
|------|----------|
| 助理 | 後台所有功能（登入後台、建菜單、彙整、薪資） |
| 員工 | LINE LIFF 點餐頁（不需登入後台） |

## 2. 後台登入

**網址：** https://lunchbot-dun.vercel.app/admin

![截圖：登入頁面](docs/images/manual-02-login.png)

1. 開啟上方網址，進入登入頁面
2. 輸入帳號（Email）與密碼
3. 點擊「登入」按鈕，進入後台首頁

> ⚠️ 帳號密碼請向系統管理員索取，不得與他人共用。

## 3. 員工名冊管理

**路徑：** 後台首頁 → 員工名冊

### 3-1. 手動新增員工

![截圖：新增員工表單](docs/images/manual-03-employee-new.png)

1. 點選右上角「新增員工」
2. 輸入員工姓名
3. 點擊「新增」按鈕
4. 新增成功後，員工出現在名冊列表中

> ⚠️ 同名員工無法重複新增，系統會提示錯誤。

### 3-2. 批次匯入員工

適合一次匯入多位員工。

![截圖：批次匯入區塊](docs/images/manual-03-employee-bulk.png)

**方法一：貼上名單**
1. 在文字框貼上員工姓名，每行一個名字
2. 點擊「匯入」

**方法二：上傳檔案**
1. 準備 `.csv` 或 `.txt` 檔，每行一個姓名
2. 點擊「選擇檔案」上傳
3. 點擊「匯入」

匯入完成後，系統會顯示「成功新增 N 人，略過重複 N 人」的摘要。

### 3-3. 刪除員工

1. 在員工列表找到要刪除的員工
2. 點擊「刪除」按鈕
3. 確認刪除

### 3-4. LINE 綁定說明

員工需綁定 LINE 帳號才能透過 LIFF 點餐。

![截圖：LINE 綁定說明區塊](docs/images/manual-03-line-binding.png)

點擊「LINE 綁定說明」展開查看五步驟綁定流程，可截圖傳給員工參考。

## 4. 店家管理

**路徑：** 後台首頁 → 店家管理

店家管理用來儲存常用店家與品項，建立菜單時可直接套用，不需每次重新輸入。

![截圖：店家管理列表（卡片式）](docs/images/manual-04-stores-list.png)

### 4-1. 新增店家

1. 點選右上角「新增店家」
2. 填入店家名稱
3. 填入品項名稱與價格（可點「新增品項」增加多筆）
4. 點擊「儲存」

![截圖：新增店家表單](docs/images/manual-04-store-new.png)

### 4-2. 編輯店家

1. 在店家卡片上點擊「編輯」
2. 修改店家名稱或品項
3. 點擊「儲存」回到列表

### 4-3. 刪除店家

**單筆刪除：** 點擊店家卡片上的「刪除」按鈕，確認後刪除。

**批次刪除：**
1. 勾選多個店家卡片左上角的核取方塊
2. 點擊上方出現的「刪除選取」按鈕
3. 確認後一次刪除

> ⚠️ 刪除店家不會影響已建立的菜單紀錄。

## 5. 菜單管理

**路徑：** 後台首頁 → 菜單管理

菜單管理用來建立每日訂餐菜單、追蹤收單狀態、推播至 LINE 群組。

![截圖：菜單管理列表（卡片式）](docs/images/manual-05-menus-list.png)

### 菜單狀態說明

| 狀態 | 說明 |
|------|------|
| 收單中 | 員工可以點餐 |
| 已結單 | 截止時間到，不再接受點餐 |
| 已刪除 | 已手動刪除 |

### 5-1. 手動建立菜單

1. 點選右上角「建立菜單」
2. 填寫以下欄位：
   - **日期**：訂餐日期
   - **餐別**（選填）：如「午餐」、「下午茶」
   - **店家名稱**
   - **截止時間**：員工最晚可以點餐的時間
   - **菜單類型**：食物店 / 飲料店
   - **品項**：名稱 + 價格（可新增多筆）
3. 若勾選「同步儲存至店家管理」，下次可直接套用為樣板
4. 點擊「建立菜單」

![截圖：建立菜單表單](docs/images/manual-05-menu-new.png)

### 5-2. 套用歷史樣板

1. 進入「建立菜單」頁面
2. 在「套用歷史樣板」下拉選單選擇店家
3. 店家名稱與品項會自動帶入
4. 補填日期與截止時間後送出

![截圖：套用歷史樣板下拉選單](docs/images/manual-05-menu-template.png)

### 5-3. 推播菜單至 LINE 群組

1. 進入菜單詳細頁（點卡片上「查看詳細」）
2. 點擊「推播至 LINE 群組」按鈕
3. 同一天收單中的菜單會合併成一則 Flex Message 推播至群組

![截圖：菜單詳細頁推播按鈕](docs/images/manual-05-menu-push.png)

### 5-4. 手動結單

截止時間到系統會自動結單，若需提前結單：

1. 進入菜單詳細頁
2. 點擊「結單」按鈕
3. 確認後狀態變更為「已結單」

### 5-5. 刪除菜單

1. 在菜單列表勾選要刪除的菜單
2. 點擊「刪除選取」
3. 確認後刪除

> ⚠️ 刪除菜單會同時刪除該菜單下的所有訂單紀錄，無法復原。

### 5-6. AI 辨識菜單（選用）

1. 進入「建立菜單」頁面，展開「AI 辨識菜單」區塊
2. 上傳菜單照片
3. 系統自動辨識品項與價格
4. 逐筆確認或修改辨識結果
5. 點擊「套用至表單」帶入品項欄位

![截圖：AI 辨識菜單區塊](docs/images/manual-05-menu-ai.png)

## 6. 助理代客點餐

適用於員工無法自行點餐（未綁定 LINE、不方便操作）的情況。

**路徑：** 菜單詳細頁 → 展開「助理代客新增/修改訂單」

![截圖：助理代客點餐區塊](docs/images/manual-06-assisted-order.png)

### 新增 / 修改訂單

1. 從下拉選單選擇員工
2. 在各品項輸入數量（留空或 0 表示不點）
3. 可填寫備註（如冰量、去骨等）
4. 點擊「送出訂單」

> 💡 同一員工重複送出會覆蓋舊訂單（upsert），不會重複計費。

### 取消訂單

在已點餐清單中，找到該員工的訂單，點擊「取消」即可。

## 7. 收單後彙整

菜單結單後，進入菜單詳細頁可查看彙整資訊。

**路徑：** 菜單管理 → 點選菜單卡片「查看詳細」

### 7-1. 店家叫貨清單

展開「店家叫貨清單」區塊，顯示：
- 各品項總數量
- 各品項小計金額
- 全部合計

![截圖：店家叫貨清單](docs/images/manual-07-store-summary.png)

點擊「推播叫貨清單至 LINE 群組」，將叫貨清單發送到公司 LINE 群組，方便聯絡店家。

### 7-2. 個人對帳清單

展開「個人對帳清單」區塊，顯示：
- 每位員工的點餐明細
- 每位員工小計金額
- 全部合計

![截圖：個人對帳清單](docs/images/manual-07-billing-summary.png)

### 7-3. 匯出 CSV

點擊「匯出對帳清單（CSV）」，下載包含以下欄位的試算表：

| 欄位 | 說明 |
|------|------|
| 員工姓名 | |
| 品項名稱 | |
| 數量 | |
| 單價 | 下單當下的價格 |
| 小計 | |
| 備註 | |
| 員工小計 | 該員工當次總金額 |

## 8. 薪資扣款

**路徑：** 後台首頁 → 薪資扣款

用來產生月結扣款紀錄，供會計核帳或匯入薪資系統。

![截圖：薪資扣款頁面](docs/images/manual-08-payroll.png)

### 8-1. 產生扣款紀錄

1. 選擇帳期（年 / 月）
2. 點擊「產生扣款紀錄」
3. 系統自動彙整該月所有已結單菜單的訂單金額，依員工分組

> 💡 若訂單在產生後有修改，重新點擊「產生扣款紀錄」會自動更新金額（只更新「待匯出」狀態的紀錄，已標記匯出的不會變動）。

### 8-2. 查看彙整表格

頁面顯示每位員工當月的：
- 點餐次數
- 總扣款金額

### 8-3. 匯出 CSV

點擊「匯出 CSV」，下載包含員工姓名與扣款金額的試算表，可直接交給會計使用。

### 8-4. 標記已匯出

確認 CSV 已交出後，點擊「標記已匯出」，避免下次重複產生時覆蓋已送出的紀錄。

## 9. 員工點餐（LIFF）

員工透過 LINE 訊息中的連結開啟 LIFF 點餐頁，無需下載 App 或登入後台。

### 9-1. 首次使用：LINE 綁定

![截圖：LIFF 身分綁定頁面](docs/images/manual-09-liff-binding.png)

1. 點擊群組訊息中的「我要點餐」連結
2. 第一次使用會看到「請選擇您的姓名」頁面
3. 從下拉選單選擇自己的姓名
4. 點擊「確認綁定」，之後開啟 LIFF 會自動識別身分

### 9-2. 點餐操作

![截圖：LIFF 點餐頁面](docs/images/manual-09-liff-order.png)

1. 點擊連結開啟點餐頁
2. 各品項以卡片呈現，點 **＋** / **－** 調整數量
3. 可在備註欄填寫特殊需求（飲料類會另有冰量 / 糖量選項）
4. 頂部即時顯示目前總金額
5. 點擊「送出訂單」完成點餐

> 💡 截止時間前可隨時重新點餐修改，重複送出會覆蓋舊訂單。截止後頁面變為唯讀，無法再修改。
